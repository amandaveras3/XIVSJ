# XIV São João Solidário — Site

## Arquivos principais
- `index.html`: estrutura do site.
- `style.css`: visual, responsividade, modo escuro e animações.
- `script.js`: idiomas, menu mobile, animações, cabeçalho inteligente e modo escuro.
- `assets/images/hero-bg.svg`: imagem de fundo da primeira página.
- `assets/images/logo.svg`: logo usada no cabeçalho.
- `assets/images/favicon.svg`: ícone da aba do navegador.
- `netlify.toml`: configuração para hospedagem gratuita no Netlify.

## Onde trocar imagens
Substitua os arquivos mantendo os mesmos nomes:
- `assets/images/hero-bg.svg`
- `assets/images/logo.svg`
- `assets/images/favicon.svg`

Se usar PNG/JPG/WEBP, atualize os caminhos no HTML e CSS.

## Onde mudar o título da aba
No `index.html`, altere:
```html
<title>XIV São João Solidário | Desv</title>
```
O JavaScript foi ajustado para não sobrescrever esse título.
